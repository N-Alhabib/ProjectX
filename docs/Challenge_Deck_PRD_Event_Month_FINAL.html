Challenge Deck PRD — Discord Server Month Event

Event: 1 month Platform: Website Audience: Discord server members
Last updated: 2026-02-03

Key Goals Tracks Auth & Sessions Daily Flow Cards & Timer Main + Quiz Hints XP & Coins Leaderboard Avatar & Champion Track Pages Out of Scope

Key Goals

  • Fun, repeatable daily routine: only 3 cards per day.
  • Fair competition: wins require correct answers (main question + quiz), not external proof.
  • Low friction: email/password login, persistent sessions (30 days).
  • Event energy: monthly leaderboard + streak bonuses + champion-track avatar decoration.
  • Learning value (for builder): clear full-stack scope (auth, DB, state, scoring, leaderboard).
This is a Discord-server event website. Members access it via a link posted in the server.

Tracks

Users must pick exactly 3 tracks each day from these 6:

Technical

  • Coding
  • SQL
  • Software Engineering
  • Web & APIs
  • Cloud Basics
  • Security Basics

Fun / Knowledge

  • Global Knowledge
  • Anime
  • Games Knowledge
  • Science (Light)
  • Movies & TV
  • Sports

Authentication & Sessions

  • Auth Gate: no content until user logs in / signs up.
  • Sign up fields: name, email (unique), password (stored hashed).
  • Persistent login: 30-day session with sliding expiration (each visit extends session).
  • Logout: available at all times.

Daily Flow

  1. User opens the site (already logged in).
  2. User must choose exactly 3 tracks.
  3. Click Generate Today’s Deck.
  4. System generates 3 random cards from the three selected tracks only, with a fairness rule: no more than 2 cards can come from the same track (no 3-of-a-kind).
  5. Example: selected (Coding, SQL, Security) → possible deck: (Coding, Coding, SQL) or (SQL, Security, Security) or (Coding, SQL, Security).
  6. Track selection is then locked for the day.
  7. At the next day, the deck expires and the user must pick tracks again.
The deck is today-only and is fixed once generated.

Cards & Timer

Start Rules

  • Card has a Preview screen.
  • To play, user presses Start and confirms: No pause / no restart / no undo.

Timer Rules

  • Timer is server-based: started_at + duration.
  • No pause.
  • User may leave the site; time continues.

Finish Button

  • Finish is available at all times during the challenge.
  • Pressing Finish shows a confirmation modal: Ending early can’t be undone.

One Active Card

  • User cannot start a second card while one is in progress.
  • After finishing (or timing out while present), user can proceed to the next card.

Verification (No Proof) — Main Question + Mini Quiz

Main Question

  • Each card contains a Main Question that must be answered within the time limit.
  • If Main is wrong → immediate fail: no quiz, XP = 0.
  • If Main is correct → unlock the Mini Quiz.

Mini Quiz

  • 3 questions, all directly related to the Main Question.
  • Pass threshold: 2/3 correct.

Card Result → XP

  • WIN: Main correct + Quiz pass (≥2/3) → XP = 3
  • PARTIAL: Main correct + Quiz fail → XP = 1
  • FAIL: Main wrong, not answered in time, or missed (below) → XP = 0

Missed Rule

  • If the timer ends while the user is away (not present on the site) → MISSED, XP = 0, no quiz.
  • Presence detection will be defined during implementation (e.g., heartbeat).

Hints

  • Hints are available after Start only (during play).
  • Daily limit: 1 hint per day total (across all cards).
  • Using the hint reveals a single helpful clue related to the Main Question.

Economy — XP, Daily Win & Streak

XP per Card Result

  • WIN → XP = 3
  • PARTIAL → XP = 1
  • FAIL → XP = 0

Daily Win

  • Daily Win is achieved only if the user gets 3 WINs that day.
  • Win Streak = consecutive days with Daily Win.

Daily Win Bonus XP

  • Base: +5 XP
  • Streak scaling: +2 XP × (streak_day_count - 1)
  • Cap: streak bonus max +20 XP/day (so max daily bonus = +25 XP)

Leaderboard (Event Month)

  • Score = Total XP (includes Daily Win bonus XP).
  • Display:
    • Top 10 view
    • All participants view (search + pagination)
  • Tie-breakers (if same score):
    1. Higher Daily Wins count
    2. Higher Win Streak
    3. Higher total Wins (XP=3 cards)

Avatar & Champion Track

  • User has an avatar (default placeholder; upload can be added later).
  • Champion Track is computed over the last 14 days:
Champion Track = track with highest WIN count (XP=3) over last 14 days
Tie-breakers:
  1) higher WIN count over last 7 days
  2) most recent WIN
  3) fixed ordering
  • Champion Track appears as a Top Decoration above the avatar (mascot vibe).
  • Mascots must be original (no copyrighted characters).

Required Pages (v1)

  1. Login / Sign up
  2. Today: pick 3 tracks → generate deck → show 3 cards
  3. Card Preview
  4. Focus Mode (timer + main question + hints)
  5. Mini Quiz
  6. Today Summary
  7. Profile (avatar, streak, champion track, basic stats)
  8. Leaderboard (Top 10 + all participants)
  9. Event Info (start/end, days left, rules)
  10. Final Results (after event ends)

Out of Scope (v1)

  • Discord OAuth / bot integration
  • Multi-server support
  • Payments
  • Full i18n (later: translate page button)
  • External proof uploads
  • Social feed / comments
